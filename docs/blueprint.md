# **App Name**: SoccerOS

## Core Features:

- Interactive Dashboard Overview: A centralized command-center dashboard displaying key club metrics, upcoming events, and player alerts with data visualization charts from mock data.
- Comprehensive Player Management: Ability to search, filter, and view detailed player profiles, including statistics, development scores, and historical data, from a mock player database.
- Team & Roster Builder: An interactive interface for assembling teams and planning rosters using drag-and-drop, complete with formation previews and player availability checks.
- AI Player Promotion & Selection Tool: Generates AI-driven recommendations for player movements between teams, academy progression, and roster suggestions based on performance and development metrics.
- Tactical & Set Piece Designer: Visual boards for designing, storing, and reviewing tactical formations, match models, and pre-defined set-piece routines for various match scenarios.
- Training Session Planner: A workflow tool to create and schedule training sessions, select drills from a library, assign objectives, and track attendance for different age groups.
- AI Video Event Tagging Tool: Upload and review match videos with AI-powered automatic tagging of key events (goals, shots, transitions) and the ability for coaches to add manual annotations and share player-specific clips.

## Style Guidelines:

- Primary UI elements and strong text: Deep Blue-Grey (#474E64) for a serious, professional, tech-oriented foundation.
- Background for content areas: Very Light Cool Gray (#F6F7FA) providing a clean, spacious backdrop for data and cards.
- Accent Color (Soccer Green): A vibrant, athletic green (#1EA048) for interactive elements, data visualization highlights, and branded calls to action.
- Dark sidebar background: A very deep, desaturated blue-black (#1C202B) to visually separate navigation from content and establish a 'command center' feel.
- Headlines and prominent display text: 'Space Grotesk' (sans-serif) for a modern, techy, and crisp feel suitable for a digital command center.
- Body text and detailed information: 'Inter' (sans-serif) for its neutral, objective readability, essential for data-heavy dashboards and player profiles.
- Utilize Lucide icons for all UI elements, thoughtfully selecting and customizing them to convey realistic and sport-specific representations where applicable, enhancing the soccer-focused professional aesthetic.
- A desktop-first, responsive layout with a fixed, dark left sidebar for consistent global navigation. Main content areas feature a spacious, clean grid of white cards on a soft gray background, emphasizing Apple-like precise spacing and clarity.
- Subtle, functional animations using Framer Motion to enhance user experience during navigation transitions, data updates within charts, and opening/closing interactive elements (e.g., drawers, modals). Avoid complex or distracting effects.
